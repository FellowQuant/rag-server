**`is_port_open(host, port, timeout)`**: stdlib socket check:
```python
import socket
def is_port_open(host: str = "localhost", port: int = 6333, timeout: float = 1.0) -> bool:
    with socket.socket(socket.AF_INET, socket.SOCK_STREAM) as s:
        s.settimeout(timeout)
        return s.connect_ex((host, port)) == 0
```

**`cmd_start_qdrant()`**: Port check first, then docker compose with embedded asset.
```python
from pathlib import Path
from importlib.resources import files
import subprocess, sys, tempfile, os

def cmd_start_qdrant() -> None:
    if is_port_open("localhost", 6333):
        print("Qdrant is already running on port 6333.")
        return
    # Write compose file to ~/.fellowquant-rag/ (persistent, avoids tempfile issues with volume paths)
    compose_dir = Path.home() / ".fellowquant-rag"
    compose_dir.mkdir(parents=True, exist_ok=True)
    compose_path = compose_dir / "docker-compose.yml"
    if not compose_path.exists():
        compose_path.write_text(
            files("rag_server.assets").joinpath("docker-compose.yml").read_text()
        )
    result = subprocess.run(
        ["docker", "compose", "-f", str(compose_path), "up", "-d", "qdrant"],
        capture_output=True, text=True
    )
    if result.returncode != 0:
        print(f"Failed to start Qdrant:\n{result.stderr}", file=sys.stderr)
        sys.exit(1)
    print("Qdrant started on port 6333.")
```

**`write_mcp_json(cwd: Path, data_dir: str) -> None`**: Read-modify-write pattern:
```python
import json
config = {}
mcp_file = cwd / ".mcp.json"
if mcp_file.exists():
    try:
        config = json.loads(mcp_file.read_text())
    except json.JSONDecodeError:
        config = {}
config.setdefault("mcpServers", {})
config["mcpServers"]["rag-server"] = {
    "type": "stdio",
    "command": "rag-server",
    "args": ["mcp"],
    "env": {"DATA_DIR": data_dir}
}
mcp_file.write_text(json.dumps(config, indent=2) + "\n")
```

**`register_mcp_global(data_dir: str) -> bool`**: Uses `claude mcp add-json` (not `claude mcp add`) to pass env block:
```python
import json, shutil, subprocess
def register_mcp_global(data_dir: str) -> bool:
    if not shutil.which("claude"):
        return False
    payload = json.dumps({
        "type": "stdio",
        "command": "rag-server",
        "args": ["mcp"],
        "env": {"DATA_DIR": data_dir}
    })
    result = subprocess.run(
        ["claude", "mcp", "add-json", "rag-server", payload, "--scope", "user"],
        capture_output=True, text=True
    )
    return result.returncode == 0
```

**`cmd_setup(first_run: bool = False)`** flow:
1. Print header: "FellowQuant RAG Server Setup" (or "First-run setup" if `first_run=True`)
2. **Scope prompt**: `safe_input("Register MCP globally (g) or locally for this project (l)? [g/l]: ", default="l")`
3. **DATA_DIR prompt**: `safe_input("Data directory [./data/]: ", default="./data/")` — strip trailing slash, default to `./data/`
4. **MCP registration (global path)**:
   - Check `claude_mcp_registered()` — if already registered, prompt `safe_input("rag-server already registered globally. Update? [y/N]: ", default="n")`
   - If yes (or not registered): call `register_mcp_global(data_dir)`
     - If `shutil.which("claude")` is None: print fallback message explaining to install Claude Code; write `.mcp.json` in CWD instead; mark as local fallback
     - If returncode != 0: print stderr, suggest manual `claude mcp add-json` command
   - On success: print "Registered globally in ~/.claude.json"
5. **MCP registration (local path)**:
   - `cwd = Path.cwd()`
   - If `mcp_json_exists(cwd)`: prompt `safe_input(".mcp.json already has rag-server. Update? [y/N]: ", default="n")`
   - If yes (or not present): call `write_mcp_json(cwd, data_dir)`
   - Print ".mcp.json written in {cwd}"
6. **llm.yaml copy**:
   - If `(cwd / "llm.yaml").exists()`: prompt `safe_input("llm.yaml already exists. Overwrite? [y/N]: ", default="n")`
   - If yes (or not present): copy from embedded asset via `importlib.resources`
   - Print "llm.yaml copied — edit it to configure your LLM provider"
7. **Sentinel write**: `SENTINEL.parent.mkdir(parents=True, exist_ok=True); SENTINEL.touch()`
8. **Summary**: print completion message listing what was done:

   ```
     Data dir:  ./data/
     llm.yaml:  ./llm.yaml (edit to configure LLM)

     rag-server start-qdrant   # start vector database

   ```
