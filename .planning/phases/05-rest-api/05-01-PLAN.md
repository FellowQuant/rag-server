**src/rag_server/api/middleware.py** — create new file:

Two BaseHTTPMiddleware subclasses:

1. `LoggingMiddleware`: records start time via time.perf_counter(), calls call_next(request), records status_code and elapsed duration in ms, logs at INFO level: `"%s %s %d %.1fms", request.method, request.url.path, response.status_code, duration_ms`. CRITICAL: never call response.body() or consume the response body — SSE streaming must pass through unmodified.

2. `UploadSizeLimitMiddleware`: accepts `max_upload_size: int` in __init__ (passed via add_middleware kwargs). In dispatch(), reads `request.headers.get("content-length")`. If present and parseable as int and exceeds max_upload_size, return JSONResponse with RFC 7807 body `{"type": "about:blank", "title": "Request Entity Too Large", "status": 413, "detail": f"Upload size {size} bytes exceeds limit of {self._max_size} bytes. Set MAX_UPLOAD_SIZE env var to increase."}` and status_code=413. If Content-Length absent or unparseable, call call_next(request) without blocking (chunked transfer allowed).

Import: `from starlette.middleware.base import BaseHTTPMiddleware` — already in fastapi/starlette 0.49.3, no new dependency.

**src/rag_server/api/errors.py** — create new file:

Define `_STATUS_TITLES` dict mapping common HTTP status codes (400, 404, 409, 413, 415, 422, 500, 503) to their standard reason phrases.

Define `_problem(status: int, detail: str) -> JSONResponse` helper that returns JSONResponse with body `{"type": "about:blank", "title": _STATUS_TITLES.get(status, "Error"), "status": status, "detail": detail}` and status_code=status.

Three async exception handler functions:
- `http_exception_handler(request, exc: StarletteHTTPException)`: extract detail as string (exc.detail if already str, else str(exc.detail)), return _problem(exc.status_code, detail)
- `validation_exception_handler(request, exc: RequestValidationError)`: join all validation errors as `"; ".join(f"{'.'.join(str(loc) for loc in e['loc'])}: {e['msg']}" for e in exc.errors())`, return _problem(422, detail)
- `unhandled_exception_handler(request, exc: Exception)`: log the exception with logger.exception(), return _problem(500, str(exc))

`register_exception_handlers(app: FastAPI)` function that calls:
- app.add_exception_handler(StarletteHTTPException, http_exception_handler)
- app.add_exception_handler(RequestValidationError, validation_exception_handler)
- app.add_exception_handler(Exception, unhandled_exception_handler)

Imports needed: `from fastapi import FastAPI`, `from fastapi.exceptions import RequestValidationError`, `from starlette.exceptions import HTTPException as StarletteHTTPException`, `from starlette.requests import Request`, `from starlette.responses import JSONResponse`.
